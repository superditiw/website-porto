Project

What've I built?

Lip Gesture-Based for Wheelchair Movement using CNN

This is a collaborative project with my friends and also served as my final thesis. Together with five of my friends, we created a motion control device for a wheelchair. On the hardware side, I assisted with soldering the wires connected to the wheelchair. For the software, I developed a mouth movement detector using Mediapipe. The dataset I used consisted of photos of my mouth, which I trained using a CNN architecture that I previously created. This project resulted in an electric wheelchair that could be controlled using the user's lip gestures while seated in the wheelchair. The benefits I gained from this project were completing my thesis, developing my soldering skills, and a better understanding of Python programming, particularly CNN in machine learning. Digital Clock Embedded System

This is my final project from the embedded system course. I made a digital clock using Arduino Uno with several sensors, namely a light sensor (LDR), a temperature sensor (LM35) and an RTC module displayed on a Dot Matrix. Some of the features displayed on the Dot Matrix include a clock, temperature, and brightness that automatically matches the room's lighting. The benefits I get from this project are that I get a new clock made by myself and I understand a little about embedded system programming using Arduino Uno. Theese are the documentations of the tool.
